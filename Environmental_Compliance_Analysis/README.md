# Drinking Water Violations Analysis

## Overview

This notebook examines drinking-water system violations at both the system and county level. The main analytical takeaway is that population size matters more than system type when interpreting violation rates, and population-weighted county metrics tell a more meaningful story than raw counts alone.

## What this project demonstrates

- Cleaning and standardizing administrative data
- Comparing groups with nonparametric statistical tests
- Modeling count outcomes with exposure-aware methods
- Translating technical output into clear written interpretation

## Dataset

- **Source:** Pennsylvania Department of Environmental Protection (DEP), Safe Drinking Water Program
- **Snapshot:** January 2026
- **File used:** `Drinking_Water_Violations.csv`

## Main findings

- Smaller systems show much higher per-capita violation rates because the denominator is small.
- System type does not show a meaningful statistical difference in this sample.
- County rankings change depending on whether simple averages or population-weighted rates are used.
- The county-level predictive model is useful as a ranking exercise, but the sample is too small for strong forecasting claims.

## Project files

- `Dwv.ipynb`: main notebook
- `Drinking_Water_Violations.csv`: source dataset used in the analysis
- `requirements.txt`: Python dependencies
- `outputs/`: saved plot files generated by the notebook
